shell中的grep

grep主要用来进行字符串数据对比，然后将符合用户需要的字符串打印出来。

1. 查找特定字符串

option description
-n 显示行在文件中的行号
-A 后面跟数字，所匹配行之后的几行也会打印
-B 后面跟数字，所匹配行之前的几行也会打印
–color 颜色标注
