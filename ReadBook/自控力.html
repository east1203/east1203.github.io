2020-06-28

01 我要做，我不要，我想要：什么是意志力？为什么意志力至关重要？

  1. 三种力量：我要做，我不要，我想要。
  2. 前额皮质与意志力有关。前额皮质并非始终可靠，醉酒、缺觉、分心等都会影响到它，使我们无法控制自己的冲动。
  3. 要意识到自己正在做什么，自己想要做什么，不要完全被意欲控制。
  4. 冥想有利于增强自控力。

02 意志力本能：人生来就有地址奶酪蛋糕的诱惑

  1. 先要意识到“不要做”什么，或者“要做”什么。做事要“三思而后行”，让自己慢下来，有利于抑制冲动。
  2. 缺乏睡眠会影响意志力。

03 累到无力抵抗：为什么自控力和肌肉一样有极限？

  1. 意志力需要消耗能量，当身体疲劳、饥饿的时候，意志力会减弱。
  2. “在一些小事上持续自控会提高整体意志力，这些小事包括改变姿势、每天用力握一个把手、戒掉甜食、记录支出情况。虽然这些小的自控力锻炼看起来无关紧要，但它却能让我们应付最关键的意志力挑战。”
  3. 疲劳不是一种身体反应，而是一种感觉，是我们大脑告诉我们需要休息，将能量留给大脑使用。
  4. 增强“想要”的力量会帮助我们克服诱惑，提高自制力。

04 容忍罪恶：为什么善行之后会有恶行？

  1. 如果我们做了一件好事，潜意识就会允许我们去做不好的事情。比如“控制住购买欲望的消费者，很可能会回家吃一些美食”。
  2. “能带给我们动力的事是我们想要的，避开我们不想要的。将某种行为道德化，使使我们对这件事的感觉变得奇怪。……如果你告诉自己锻炼、存钱或戒烟是正确的事，而不是件能让你达成目标的事，你就不太可能持之以恒了。”
  3. “不要把支持目标的行为误认为目标本身，不是说你做了一件和你的目标一致的事情，你就不会再面临危险了，注意观察一下，你是否因为某些积极的行为值得称赞，就忘记了你的目标。”
  4. “大脑会对完成目标的可能性感到兴奋，会将可能性看成已经完成了那个目标，所以及时行乐就会成为首选。比如我们做了健康的事物，就会对不健康的食物产生强烈的渴望。
  5. “我们之所以会奖励自己的良好行为，是因为我们在内心深处认为，真正的自己想要做坏事。从这一点来看，每一次做好事都是惩罚，每一次做坏事都是奖励。……想要走出”道德许可”的陷阱，我们就要知道，那个想变好的自己才是真正的自己。“

05 大脑的逆天大谎：为什么我们误把渴望当成幸福？

  1. 大脑中的奖励系统驱动我们采取行动和消耗体能。每当这个区域受到刺激的时候，大脑就会说：“再来一次，这会让你感觉良好！”奖励系统是承诺给我们快感，诱惑我们采取下一次行动，而不是这个行动真正给我带来快感。

  2. 我们知道可能受到新的信息，或者下一个视频会让我们开怀大笑，所以就会在APP上不停刷新。这是受到了可能得到快感的承诺（奖励）的诱惑。

  3. “当奖励的承诺释放多巴胺的时候，我们更容易受到其他形式的诱惑。比如，色情图片更容易使男性在经济上冒险。“

  4. 商店会通过音乐、图片、免费试吃、漂亮的售货员等来刺激你的奖励系统，让其释放更多的多巴胺。让你买更多可能不需要的东西。

  5. 大脑的奖励系统对新鲜感和多样性也会有反应，你的多巴胺神经元会对熟悉的奖励反应较少，即使那是让你真正获得享受的东西，……零售商会为压箱底的旧款推出新的颜色。

  6. ”当人们主意到奖励的承诺不过是假象时，魔咒就解开了。如果你强迫你的大脑，让它一边期待着奖励——获得快乐、幸福，或者结束痛苦，一边真切地感受现实状况，那么大脑最终会调整它的期望值。“

  7. “当他们不再需要快感的时候，他们就失去了动力，如果你想不出任何一件让你感觉良好的事，你就很难从床上爬起来做事。这种毫无欲望的状态耗尽了希望，也夺走了很多人的生命。”

06 “那又如何”：情绪低落为何会使人屈服于诱惑

  1. ”当你情绪低落的时候，你会怎么样让你高兴起来吗？如果你和大多数人一样，你就会选择奖励的承诺。“
  2. “奖励的承诺并不总意味着我们会得到快了，通常，我们环节压力的方法反而会让我们更有压力。”
  3. “大脑不仅会保护人的声明，还会维持人的心情。所以，当你感到压力的时候，你的大脑会指引着你，让你去做它认为能带给你快乐的事情。”
  4. “恐惧管理”——烟盒上的警告会提高烟民抽烟的欲望。死亡的警告会让烟民感到压力和恐惧，这种压力会让烟民采用默认的方法缓解压力——抽烟。
  5. “当你（又一次）责备自己（又一次）屈服于诱惑的时候，往往会带来更多意志力的失败，造成更多的痛苦。但是，你寻求安慰的东西并不能中断这个循环，它只会给你带来更深切的罪恶感。”
  6. 犯错之后的“那又如何”想法会让我们更加放纵自己，做更多不好的事情。
  7. “自我批评会降低积极性和自控力，而且也是最容易导致抑郁的因素，它不仅耗尽了”我要做“的力量，还好近了”我想要“的力量。”
  8. “在个人挫折面前，持自我同情态度的人比持自我批评态度的人更愿意承担责任，他们也更愿意接受别人的反馈和建议，更可能从这种经历中学得到东西。”

07 出售未来：及时享乐的经济学

  1. “10分钟法则”——做某事之前延迟10分钟 ，那么做这件事的欲望就会减弱。
  2. 将充满诱惑的事与自己的目标建立联系，想象一下，如果我做了这件事，我就不能达到我的目标，这样值得吗？（自己）
  3. 很多时候，我们对未来的自己期望过高，把现在该做或者不想做的事情推到未来去做，但是未来的自己跟现在的自己真的有那么多的进步吗？不见得。可能被拖延的事情我们最终还是做了，但是之所以会去做，很可能不是因为此时我们的能力增强很多，而是因为deadline到了，我们不得不做。(自己)
  4. 我们对未来的自己期望过高是导致拖延症的重要原因。（自己）

08 传染：为什么意志力会传染？

  1. 我们的行为、目标、意志力会受到周围环境、人的传染。
  2. “社会认同”；“别人都这么做”
  3. “羞愧作为一种预防措施或许能起作用。但事情一旦结束，羞愧就更可能引起自我伤害，而非自我控制。”
  4. “自豪的力量”——人们在公共场所更容易购买绿色产品。

09 别读这章：“我不要”力量的局限性

  1. 大脑里没有处理“我不要”做什么的机制。比如你告诉自己不要想北极熊，但你的脑海里反而会出现北极熊的图像。
  2. “其他学生预先了解了”白熊“现象，实验组织者向他们解释了”反弹“理论，让他们不要强制自己忘掉吃巧克力的想法。相反，当他们发现自己想要吃巧克力时，接受自己的想法和感受，但同时也要记住，不要顺着自己的想法去做。即使不试着控制自己的思维，也要控制自己的行为。”
